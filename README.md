## Features-產品功能
1.使用者可依照餐廳名或餐廳類別搜尋
2.點入餐廳頁面可檢視餐廳的詳細資訊：餐廳名、類別、電話、地址、Google Map簡介
3.點擊餐廳類別可檢視相同類別的餐廳
## Environment SetUp- 環境建置
Node.js
## Installation-安裝
1.打開終端機，clone專案到本機
```
git clone https://github.com/SeijoHuang/restaurant_list.git
```
2.進入專案資料夾
```
cd restaurant_list
```
3.載入相關套件，終端機輸入
```
npm install
```
4.啟動伺服器，終端機輸入
```
npm run dev
```
